# Section 1: Elixir Fundamentals

Welcome to Elixir! This section covers the core concepts you need to start writing Elixir code.

## What You'll Learn

- Basic types and data structures
- Pattern matching - Elixir's superpower
- Control flow constructs
- Functions and modules
- The pipe operator

## Prerequisites

- Elixir 1.15+ installed
- A code editor
- Basic programming experience in any language

## Running Lessons

```bash
# Run a lesson and see output
elixir 01_hello_elixir.exs

# Run interactively in IEx
iex 01_hello_elixir.exs
```

## Tips for Learning

1. **Type everything** - Don't copy/paste, type the examples
2. **Experiment in IEx** - Try variations of each example
3. **Break things** - See what errors look like
4. **Complete exercises** - They reinforce learning
5. **Take notes** - Write down what surprises you
